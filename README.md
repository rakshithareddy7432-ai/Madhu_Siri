# 🐝 MadhuSiri – Smart Bee Protection and Farmer Alert System

## 📌 Project Description
MadhuSiri is an Android application developed to help farmers and beekeepers work safely together. The app provides spray alerts, hive management, bee health monitoring, Google Maps location tracking, and safety tips to protect honey bees from harmful pesticide spraying.

---

## 🚀 Features

### 🌾 Farmer Features
- View Farm Location on Google Maps
- Spray Alert Information
- Pesticide Suggestions
- Safe Spraying Tips

### 🐝 Beekeeper Features
- Add Hive Details
- Save Hive Location
- Monitor Bee Health
- Safety Guidance Tips

---

## 🎯 Objective
The objective of MadhuSiri is to improve coordination between farmers and beekeepers and reduce the harmful effects of pesticides on honey bees.
